# Инструкция по развёртке проекта
1. Импорт базы данных
- Скачать на пк AptekaHubDB.bacpac
- Открыть SQL Server Management Studio и совершить соединение к серверу
- Нажать ПКМ на файл «Базы данных» и нажать «Импорт приложения уровня данных» 
- В «Параметры импорта» нажимаем на «Обзор» и выбираем AptekaHubDB.bacpac. 
- Несколько раз нажимаем «Далее» и «Готово». Ждём, пока наша БД не импортируется.
2. Настройка приложения
- Скачать и распаковать AptekaInternetApp.zip
-  В файлах проекта находим ContextDB.cs, (путь файла AptekaInternetApp\ AptekaInternetApp\Models\ContextDB.cs) вместо HOME-PC вводим название сервера, в котором находится наша база данных.
- Сохраняем.

Поздравляю! Вы успешно развернули проект и готов к использованию.
В случае ошибок стоит проверить соответствия названий файлов и действий, согласно инструкции.
